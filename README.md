# MWD 集點卡系統（QR 自助、無實體卡、不用 POS、不用 LINE）

一套免費的數位集點卡：客人掃一張固定 QR Code，輸入手機號碼當帳號，再輸入「今日通關碼」即可自助蓋章。點數存在雲端（Google 試算表），**不怕卡片遺失**，店員平常幾乎不用操作。

## 特色

- **客人自助**：掃 QR → 輸入手機 → 輸入今日通關碼 → +1 點，全程客人自己完成。
- **不怕遺失**：點數綁手機號碼存雲端，沒有實體卡可掉；換手機也找得回。
- **店員幾乎零操作**：每天早上換一次「今日通關碼」、兌換時確認一次，平常不插手。
- **防作弊**：加點需「今日通關碼」＋「每支手機每天限蓋一次」。
- **零成本**：用 Google 試算表 + Apps Script，免費、Google 代管，不需伺服器、不需 POS、不需 LINE。

## 運作流程

1. 牆上／櫃台貼一張**固定不變的 QR**（連到部署好的網頁）。
2. 客人掃 QR → 第一次輸入手機號碼（之後自動記得）。
3. 畫面顯示目前點數與 10 格進度圖。
4. 客人輸入櫃台小白板上的「今日通關碼」→ 按蓋章 → +1。
5. 集滿 10 點 → 出示兌換畫面 → 店員輸入兌換碼 → 點數歸零。

```

│   └── appsscript.json   # Apps Script 設定檔

│   └── 部署教學.md        # 一步一步部署說明
```

## 快速開始

詳見 [docs/部署教學.md](docs/部署教學.md)。重點：

1. 建一個 Google 試算表 → 擴充功能 → Apps Script。
2. 把 `apps-script/` 三個檔案內容貼進去。
3. 部署成「網頁應用程式」，存取權設「任何人」。
4. 把網址轉成 QR Code，印出來貼櫃台。
5. 每天早上到 `Config` 工作表改「todayCode」。

## 集點規則（預設，可改）

- 每筆消費蓋 1 點（每支手機每天限一次）。
- 集滿 10 點，換指定餐點或飲料一份。
- 規則參數都在 `Code.gs` 最上方的 `CFG` 與試算表 `Config` 工作表，可自行調整。
